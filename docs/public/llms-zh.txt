## Agent 工作流

- CLI 模式是 Codex、Claude 等 agent 会话的推荐工作流
- MCP 仍然是 `1mcp serve` 背后的后端互操作层
- CLI 模式把 agent 侧流程从直接暴露整个 MCP 工具面改成渐进式披露
- 对于同一个 agent，只能选择一种模式；切换到 CLI 模式时，要移除该 agent 的直接 MCP server 配置
- 先运行 `1mcp instructions` 获取当前 CLI playbook 和服务器清单
- 在调用工具前，先执行 `1mcp inspect <server>` 与 `1mcp inspect <server>/<tool>`
- 使用 `1mcp run <server>/<tool> --args '<json>'` 执行选中的工具
- 使用 `1mcp cli-setup --codex` 或 `1mcp cli-setup --claude` 为支持的 CLI 安装引导文档和 hooks
